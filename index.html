03

วางแผนอ่านให้คุ้ม — วิชาที่ใช้ร่วมกัน

ข่าวดีคือ 3 คณะนี้ใช้วิชาทับซ้อนกันเยอะ ถ้าวางแผนดี อ่านทีเดียวยื่นได้หลายที่ นี่คือวิชาที่ "คุ้มสุด" เพราะใช้ได้หลายคณะ

A-Level คณิต 1

ใช้ทั้ง 3 คณะ! — CS (50%), จิตวิทยา รูปแบบ A (20%), นิติ (วิชาเลือก 20%) ถ้าเก็บวิชานี้แน่น คือลงทุนครั้งเดียวคุ้ม 3 ต่อ

TGAT (3 พาร์ต)

ใช้ทั้ง 3 คณะ — CS (35%), จิตวิทยา รูปแบบ B (100%), นิติ (50%) เป็นวิชาฟรีค่าสมัครและคืนคะแนนสูงทุกที่ ควรทุ่มแต่เนิ่น ๆ

A-Level สังคม / ไทย / อังกฤษ

ใช้ทั้งจิตวิทยา (รูปแบบ A อย่างละ 20%) และนิติ (อย่างละ 10%) — เก็บ 3 วิชานี้ครอบคลุมทั้งสองคณะสายสังคมพร้อมกัน

A-Level ชีววิทยา

เฉพาะจิตวิทยา รูปแบบ A (20%) — แต่น้องสายวิทย์เรียนอยู่แล้ว ถือเป็นแต้มต่อที่เด็กสายอื่นไม่มี

🧠
แผนรวบยอดสำหรับน้อง: ทุ่ม TGAT + A-Level คณิต 1 ให้แน่นก่อน (สองตัวนี้ครอบทั้ง 3 คณะ) แล้วเสริม ชีวะ + สังคม + ไทย + อังกฤษ = ครบทุกวิชาที่ต้องใช้ของทั้งสามเป้าหมาย ยื่นรอบ 3 ได้ทั้ง CS, จิตวิทยา และนิติ ในการเตรียมตัวชุดเดียว
